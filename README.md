---

## 📌 Course Overview & Syllabus

This course provides a comprehensive journey through information retrieval systems, moving from foundational keyword-based indexing to state-of-the-art neural architectures, generative search, and multi-modal retrieval.

### 🔹 Phase 1: Foundations & Web IR
* **Core Operations:** Text preprocessing, tolerant retrieval, and large-scale indexing.
* **Retrieval Models:** Vector Space Model, efficient document scoring, and ranking.
* **Web & Evaluation:** Web crawling architectures, near-duplicate page detection, and IR system evaluation.

### 🔹 Phase 2: Machine Learning & Text Mining
* **Probabilistic Approaches:** Probabilistic IR frameworks and n-gram language models.
* **Text Processing:** Supervised text classification, unsupervised document clustering, and word representations.
* **Neural Foundation:** Introduction to Neural Networks and Learning to Rank (LTR) algorithms.

### 🔹 Phase 3: Deep Learning & Advanced IR
* **Generative & Agentic AI:** Transformers, deep retrieval models, Retrieval-Augmented Generation (RAG & RAGAS), and Agentic IR systems.
* **Downstream Applications:** User personalization systems, Recommendation Systems, and Multi-modal IR.
